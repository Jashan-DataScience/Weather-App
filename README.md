# Weather App 🌦️

A simple, responsive web application that provides real-time weather information for any city in the world.

## Features
- Search for weather by city name.
- Displays current temperature in Celsius.
- Shows weather conditions (e.g., clear, clouds, rain).
- Responsive design for different screen sizes.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)

## Live Demo
Check out the live project here: [https://jashan-datascience.github.io/Weather-App/](https://jashan-datascience.github.io/Weather-App/)

## How to Run Locally
1. Clone this repository.
2. Open `index.html` in your web browser.
3. Note: You will need your own API Key from OpenWeatherMap to make the API calls work.
